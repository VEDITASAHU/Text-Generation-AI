# 🧠 LSTM Text Generation Model

## 📌 Overview
This project builds a deep learning model using LSTM (Long Short-Term Memory) networks to generate text. The model is trained on a news dataset and learns to predict the next word in a sequence, allowing it to generate human-like sentences.

---

## 🚀 Features
- Text preprocessing using tokenization (NLTK)
- Sequence-based next word prediction
- LSTM neural network for text generation
- Customizable text generation with creativity parameter
- Model saving and loading for reuse

---

## 🛠️ Tech Stack
- Python  
- TensorFlow / Keras  
- NumPy  
- Pandas  
- NLTK
